# ⚙ Installation & Setup

If you want to run this project locally:

### Clone the repository

```
git clone https://github.com/lucifer13598/my-portfolio.git
```

### Go to project folder

```
cd my-portfolio
```

### Install dependencies

```
npm install
```

### Run development server

```
npm run dev
```

Now open:

```
http://localhost:3000
```
